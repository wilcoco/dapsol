# 서비스 개요 (Service Overview)

## 핵심 비전

**집단지성(Collective Intelligence)**은 AI 질문-답변을 씨앗으로, 인간의 협업 투자와 검증을 통해 살아있는 지식 생태계를 키워나가는 플랫폼이다.

기존 AI 챗봇과의 차별점:
- **1회성 답변이 아닌 진화하는 지식**: Q&A가 공유되면 다른 사용자가 개선, 확장, 반박할 수 있음
- **경제적 인센티브로 품질 보증**: 신뢰 포인트(Trust Point) 투자 → 좋은 Q&A를 먼저 발굴한 사람이 보상
- **지식 그래프로 연결**: 개별 Q&A가 클러스터(마을)를 형성하고, 클러스터 간 관계로 문명 지도를 구성

## 문명 메타포

| 용어 | 영어 | 의미 |
|------|------|------|
| 영토 | Territory | 개별 QASet (질문-답변 세트) |
| 개척 | Pioneer | 새로운 질문을 시작하는 행위 |
| 경작 | Cultivate | 긍정 투자 (이 Q&A가 가치 있다고 믿음) |
| 사냥 | Hunt | 부정 투자 (오류 발견, 품질 문제 지적) |
| 마을 | Village | TopicCluster (유사한 Q&A들의 묶음) |
| 문명 지도 | Civilization Map | ClusterRelation (마을 간 관계 네트워크) |

## 핵심 플로우

```
1. 검색 → 기존 Q&A 탐색 (시맨틱 + 키워드)
2. 개척 → 새 질문 시작 → AI 대화 → 멀티턴 Q&A
3. 공유 → Q&A를 공유 풀에 게시
   └─ 백그라운드: 키워드 생성 → 임베딩 → 지식카드 추출 → 자동 링크 → 클러스터 배정
4. 경작/사냥 → 다른 사용자가 투자
   └─ 50% 품질풀, 40% 선투자자 보상, 10% 소각
5. 확장/분기 → 기존 Q&A를 기반으로 새 대화 시작
```

## 🤖→👤 AI가 인간에게 질문

AI가 답변 중 불확실한 영역을 감지하면, 해당 분야 전문가(기여 이력 기반)에게 직접 질문하는 역방향 흐름.
- `KnowledgeGap` 모델로 추적
- `Message.gapQuestionId`, `Message.isGapResponse`로 연결

## 대상 사용자

한국어 사용자. UI, 관계 라벨, 사냥 사유 등 모든 사용자 대면 텍스트가 한국어.
코드 주석은 한국어/영어 혼용.
