## 핵심 차별점

### 기존 (탐험 모드)
- "모르는 것을 AI에게 물어보고 검증받는다"
- 사용자 = 답을 찾는 사람
- 가치 = AI 답변 + 사람 검증

### 새로운 (전문가 모드)
- "AI가 모르는 것을 찾아서 내가 채운다"
- 사용자 = 답을 아는 사람
- 가치 = **내가 추가한 부분** (AI는 조연)

---

## 핵심 게임 루프

```
1. AI가 약할 만한 질문 찾기 (사냥)
         ↓
2. AI한테 시켜보기 (테스트)
         ↓
3. 부족한 부분에 내 지식 추가 (빛나기)
         ↓
4. 공유 → 다른 사람 평가 (발자국)
         ↓
5. 내 지식 자산 증가 (보상)
```

---

## AI가 약한 영역 (타겟)

| 영역 | 예시 |
|------|------|
| 로컬/현장 정보 | 우리 동네 숨은 맛집, 최근 바뀐 것 |
| 실제 경험 | 면접 후기, 제품 실사용기 |
| 삽질 기록 | 이 에러 해결법, 이 설정 방법 |
| 비교/선택 | A vs B 실제로 써보니 |
| 생활 노하우 | 자취 꿀팁, 육아 경험 |
| 현장 이야기 | 여행 중 겪은 일, 실시간 상황 |

## 수익화 모델

### 광고 발자국 모델
```
광고주 → 광고 콘텐츠(질문 형태) → 사용자 의견/후기 → 발자국 보상
                                         ↓
                              검증된 사용자만 "실효 발자국" 인정
```

### 실효 발자국 (파밍 방지)
```
실효 발자국 = 기본 발자국 × 신뢰 계수

신뢰 계수 구성:
- 휴대폰 인증: +0.5
- 실명 인증: +0.3
- 받은 발자국: +0.1~0.5
- Hub 점수: ×1.0~1.5
- 계정 나이: +0.1~0.3
```

### 수익 분배
```
광고주 100만원 집행
    ↓
├── 플랫폼 수수료: 30만원 (30%)
└── 사용자 풀: 70만원 (70%)
        ↓
    실효 발자국 비율로 분배
```
